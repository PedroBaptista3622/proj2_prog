Os programas cumprem ambos maioritariamente as especificações fornecidas.
Listam-se de seguida as exceções:
   - No programa cwcreator, este não verifica as combinações casuais de
  letras, por exemplo no caso de serem colocadas duas palavras lado a lado.
   + Ambos os programas suportam tabuleiros muito grandes (até 702 filas).
   + Suporte razoável multiplataforma (instruções variáveis, ambas as
  implementações da biblioteca de cor...)
